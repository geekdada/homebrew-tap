# Homebrew Tap

Homebrew formulae for [geekdada](https://github.com/geekdada) projects.

## Installation

```sh
brew tap geekdada/tap
brew install models
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `models` | Terminal UI for browsing AI model data from [models.dev](https://models.dev) |
